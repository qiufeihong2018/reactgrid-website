## Idea of handling data changes

1. User try to change data inside cell in one of few ways (typing new value, fill handle, copy and paste, click toggle
   icon)
2. ReactGrid event (`onCellsChanged`) is called
3. Your handler function handles event and updates state of application
4. `ReactGrid` component is rerendered with new data

Keep in mind that if you don't handle `onCellsChanged` event, your grid is readonly.

## Sample handling function

Let's declare our handling function:

```tsx
const handleChanges = (changes: CellChange[]) => {
  let newState = { ...state }
  changes.forEach((change: CellChange) => {
    const changeRowIdx = newState.rows.findIndex(
      el => el.rowId === change.rowId
    )
    const changeColumnIdx = newState.columns.findIndex(
      el => el.columnId === change.columnId
    )
    newState.rows[changeRowIdx].cells[changeColumnIdx] = change.newCell
  })
  setState(newState)
  return true
}
```

... then add our handler function to `ReactGrid` component props:

```tsx
return (
  <ReactGrid
    rows={state.rows}
    columns={state.columns}
    license={"non-commercial"}
    onCellsChanged={handleChanges} // highlight-line
  />
)
```
