## About keyboard

ReactGrid allows you to interact with grid using keyboard shortcuts in a way known from other spreadsheet like apps. 

At this moment we plan to add few callback events for clipboard action like copying, pasting and cutting.

## Keyboard shortcuts

| Shortcut                              | Action        |                    
| ------------                          | ------------- |
| <kbd>↑</kbd>   | Moves cell focus cell up one cell (if is focusable) | 
| <kbd>↓</kbd>   | Moves cell focus cell down one (if is focusable) | 
| <kbd>←</kbd>   | Moves cell focus cell to the left (if is focusable) | 
| <kbd>→</kbd>   | Moves cell focus cell to the right (if is focusable) | 
| <kbd>Ctrl</kbd> + <kbd>Space</kbd>  | Expands selection in vertical direction (only on ) | 
| <kbd>Shift</kbd> + <kbd>⌘</kbd> + <kbd>Home</kbd> or <kbd>Shift</kbd> + <kbd>⌘</kbd> + <kbd>Home</kbd>  | Selects all cells above currect selection | 
| <kbd>Shift</kbd> + <kbd>⌘</kbd> + <kbd>End</kbd> or <kbd>Shift</kbd> + <kbd>⌘</kbd> + <kbd>End</kbd>  | Selects all cells below currect selection | 
| <kbd>⌘</kbd> + <kbd>A</kbd> or <kbd>Ctrl</kbd> + <kbd>A</kbd>  | Selects all cells | 
| <kbd>⌘</kbd> + <kbd>Home</kbd> or <kbd>Ctrl</kbd> + <kbd>Home</kbd>  | Selects top left cell | 
| <kbd>⌘</kbd> + <kbd>End</kbd> or <kbd>Ctrl</kbd> + <kbd>End</kbd>  | Selects bottom right cell | 
| <kbd>Ctrl</kbd> + <kbd>End</kbd>  | Select whole column on Windows | 
| <kbd>Shift</kbd> + <kbd>Enter</kbd>   | Closes cell editor if is opened and moves focus cell above lastly edited cell | 
| <kbd>Shift</kbd> + <kbd>↑</kbd>  | Selects one cell above | 
| <kbd>Shift</kbd> + <kbd>↓</kbd>  | Selects one cell below | 
| <kbd>Shift</kbd> + <kbd>←</kbd>  | Selects one cell of the left | 
| <kbd>Shift</kbd> + <kbd>→</kbd>  | Selects one cell of the right | 
| <kbd>Shift</kbd> + <kbd>Tab</kbd>  | Moves focus to the left if single cell is selected or moves focus to the left inside range selection  | 
| <kbd>Shift</kbd> + <kbd>Enter</kbd>  | Moves focus to cell above if single cell is selected or moves focus up inside range selection | 
| <kbd>Shift</kbd> + <kbd>Space</kbd>  | Expands selection in vertical direction (from top to bottom)| 
| <kbd>Shift</kbd> + <kbd>Home</kbd>  | Expands selection to the first column | 
| <kbd>Shift</kbd> + <kbd>End</kbd>  | Expands selection to the last column | 
| <kbd>Shift</kbd> + <kbd>PageUp</kbd>  | Increase or decrease selection by single page | 
| <kbd>Shift</kbd> + <kbd>PageDown</kbd>  | Increase or decrease selection by single page | 

| <kbd>Enter</kbd> | Opens cell editor if cell if focused or moves focus down inside range selection | 
| <kbd>Delete</kbd> or <kbd>Backspace</kbd>  | Wipe selected range or only focused location | 
| <kbd>↑</kbd> | Moves focus up (if cell is focusable) | 
| <kbd>↓</kbd> | Moves focus down (if cell is focusable) | 
| <kbd>←</kbd> | Moves focus to the left (if cell is focusable) | 
| <kbd>→</kbd> | Moves focus to the right (if cell is focusable) | 
| <kbd>Tab</kbd> |  Moves focus to the right if single cell is selected or moves focus to the right inside range selection | 
| <kbd>Home</kbd> | Moves focus to the first column | 
| <kbd>End</kbd> | Moves focus to the last column | 
| <kbd>PageUp</kbd> | Moves focus up by one page (visible number of rows) | 
| <kbd>PageDown</kbd> | Moves focus up by one page (visible number of rows) | 
| <kbd>Escape</kbd> | Aborts cell edition with without changing it |
