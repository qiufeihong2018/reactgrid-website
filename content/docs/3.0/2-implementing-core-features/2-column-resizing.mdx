## How to implement column resizing?

1. For each column which should be resizable, add the `resizable` property
to the corresponding column object and set its value to `true`.

```tsx
// ...
columns: [
    // columns can now be resized
    { columnId: "Name", resizable: true, width: 100 }, //highlight-line
    { columnId: "Surname", resizable: true, width: 100 } //highlight-line
],
// ...
```

If you hover over the vertical border line between column headers now,
you will see your mouse pointer change to indicate that a column can be resized.
But you'll quickly notice that the column resizing functionality doesn't work.
Why is that? We still need to handle the events fired by ReactGrid when a column is resized.
Similar to how we handled data changes in [Handling data changes](link), we can also
define a handler function for that and pass it to our ReactGrid component.

2. Implement the event handler function

```tsx
const handleColumnResize = (ci: Id, newWidth: number) => {
  const newState = { ...state };
  const resizedColumn = newState.columns.find(
    column => column.columnId === ci
  );
  resizedColumn.width = newWidth;
  setState(newState);
}
```

3. Pass the handler function to your `ReactGrid`

```tsx
return (
  <ReactGrid
    rows={state.rows}
    columns={state.columns}
    license={"non-commercial"}
    onColumnResized={handleColumnResize} //highlight-line
  />
)
```
