## Introduction

ReactGrid allows you to "freeze" chosen rows and columns at the top or bottom and to the left or right sides
of the ReactGrid viewport. A frozen row or column will be visible at all times, no matter the scroll position.

Frozen rows and columns can be enabled by adding dedicated `ReactGrid` component props:
- `frozenTopRows`
- `frozenBottomRows`
- `frozenLeftColumns`
- `frozenRightColumns`

All of these properties are numeric and determine the number of columns or rows frozen next to the grid's edges.

## How to freeze columns and rows?

This guide is based on [getting started](link).

1. Add some more data to the table, so that it's more likely for a scrollbar to appear

```tsx

	{ columnId: "Occupation", width: 230 } // highlight-line

		rowId: 0,

			{ type: "header", text: "Occupation" } // highlight-line
		]
	},
	{
		rowId: 1,
		cells: [
			{ type: "text", text: "Thomas Anthony" },
			{ type: "text", text: "Goldman" },
			{ type: "date", date: new Date("1989-04-01") }, // highlight-line

			{ type: "text", text: "Textile and furnishings occupation" } // highlight-line
		]

```

2. Use the properties mentioned in the introduction to set the number of frozen panes at each edge

```tsx
// ...
return (
	<ReactGrid
		rows={state.rows}
		columns={state.columns}
		license={"non-commercial"}
		frozenLeftColumns={1} // highlight-line
		frozenRightColumns={1} // highlight-line
		frozenTopRows={1} // highlight-line
		frozenBottomRows={1} // highlight-line
	/>
);
// ...
```
