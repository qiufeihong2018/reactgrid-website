Highlight is an element to mark any cell inside grid.

Highlight (in oppesed to *focus*):
- can be placed on any cell
- unlike to *focus* can't be managed by user


## Definition

```ts
interface Highlight {
    readonly rowId: Id;
    readonly columnId: Id;
    readonly borderColor?: string;
    readonly className?: string;
}
```

## Properties

| Property name     | Type                                      | Property description |
| -----------       | ------                                    | -------------------- |
| `rowId`           | [`Id`](/docs/3.0/7-api/1-types/4-id/)     | Row id of cell to highlight  |
| `columnId`        | [`Id`](/docs/3.0/7-api/1-types/4-id/)     | Column id of cell to highlight  |
| `borderColor`     | `string`                                  | Optional border color |
| `className`       | `string`                                  | Optional CSS classname of highlight element  |
