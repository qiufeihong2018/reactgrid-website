`CellLocation` is an interface that describes focus position inside grid.

## Definition

```tsx
interface CellLocation {
    readonly rowId: Id;
    readonly columnId: Id;
    readonly color?: string;
}
```

## Properties

| Property name | Type          | Property description  |
| -----------   | -----         | --------- |
| `rowId`       | [`Id`](link)  | Row of cell to focus  |
| `columnId`    | [`Id`](link)  | Column of focused cell  |
| `color?`      | `string`      | Name of CSS class  |
