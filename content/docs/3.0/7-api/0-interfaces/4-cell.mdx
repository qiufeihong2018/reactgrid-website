`Cell` interface is a base for built-in cell templates and your own, it has only one necessary prop - `type`.

#### Definition

```ts
interface Cell {
    type: string;
    style?: CellStyle;
    className?: string;
}
```

#### Properties

| Properties     | Type                                                         | Properties description |
| -----------    | ---------                                                    | ---------- |
| `type`         | `string`                                                     | Name of cell type, must be unique |
| `style?`       | [`CellStyle`](/docs/3.0/7-api/0-interfaces/7-cell-style/)    | Allowed style properties contained in [`CellStyle`](/docs/3.0/7-api/0-interfaces/7-cell-style/) interface |
| `className?`   | `string`                                                     | Additional CSS classes |
