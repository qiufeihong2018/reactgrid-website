`Id` is a common type to identify many objects in grid e.g.: 
- [`Highlight`](/docs/3.0/7-api/0-interfaces/9-highlight/), 
- [`CellLocation`](/docs/3.0/7-api/0-interfaces/6-cell-location/),
- [`CellChange`](/docs/3.0/7-api/1-types/2-cell-change/),  
- [`Column`](/docs/3.0/7-api/0-interfaces/3-column/), 
- [`Row`](/docs/3.0/7-api/0-interfaces/2-row/)

`Id` can be a `number` or `string`.

#### Definition

```ts
type Id = number | string;
```
