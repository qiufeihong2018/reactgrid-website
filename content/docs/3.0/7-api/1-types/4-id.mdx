`Id` is a common type to identify many objects in grid e.g.: 
- [`Highlight`](link), 
- [`CellLocation`](link),  
- [`CellChange`](link),  
- [`Column`](link), 
- [`Row`](link)

`Id` can be an `number` or `string`.

## Definition

```tsx
type Id = number | string;
```
