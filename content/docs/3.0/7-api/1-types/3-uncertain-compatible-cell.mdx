`UncertainCompatible` is a cell type that is compatible beetwen other cell types, 
but they can be intances of various types (e.g. `DataCell` and `TimeCell`).

## Definition

```tsx
type UncertainCompatible<TCell extends Cell> = Uncertain<TCell> & {
    text: string;
    value: number;
}
```
