ReactGrid public API contains an union of basic [`Cell`](link)`[]` usually used for consuming changes and 
marking cells array inside data row.

## Definition

```tsx
type DefaultCellTypes = CheckboxCell | DateCell | EmailCell | GroupCell | HeaderCell | NumberCell | TextCell | TimeCell;
```
