## Idea of cell templates

In [`getting started`](link) chapter we declared cells array as part of ReactGrid property.

Each cell template has own unique behavior, content formating, styling. 

Nothing prevents you for creating your custom cell - read about [creating your own cell template](link). 

In listing below we declared two cell of `header` type, and they are related to interface of [`HeaderCell`](link) 
template.

```tsx
cells: [
    { type: "header", text: "Name" },
    { type: "header", text: "Surname" }
]
```

Every cell object must contain `type` attribute, other keys depends on exact type of cell.
