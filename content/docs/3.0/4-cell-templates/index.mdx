## Idea of cell templates

In [`getting started`](/docs/3.0/1-getting-started/) chapter we declared cells array as part of ReactGrid property.

Each cell template has own unique behavior, content formating and styling. 

Nothing prevents you for creating your custom cell - read about [creating your own cell template](/docs/3.0/5-create-your-own-cell-template/). 

In listing below we declared two cell of `header` type, and they are related to interface of [`HeaderCell`](/docs/3.0/4-cell-templates/6-HeaderCell/) 
template.

```ts
cells: [
    { type: "header", text: "Name" },
    { type: "header", text: "Surname" }
]
```

Every cell object must contain `type` attribute, other keys depends on exact type of specyfic cell.
