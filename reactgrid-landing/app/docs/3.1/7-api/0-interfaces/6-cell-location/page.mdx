`CellLocation` is an interface that describes position inside the grid.

#### Definition

```ts
interface CellLocation {
    readonly rowId: Id;
    readonly columnId: Id;
}
```

#### Properties

| Property name | Type                                      | Property description          |
| -----------   | -----                                     | ---------                     |
| `rowId`       | [`Id`](/docs/3.1/7-api/1-types/4-id/)     | Row [`Id`](/docs/3.1/7-api/1-types/4-id/) of cell location       |
| `columnId`    | [`Id`](/docs/3.1/7-api/1-types/4-id/)     | Column [`Id`](/docs/3.1/7-api/1-types/4-id/) of cell location    |
