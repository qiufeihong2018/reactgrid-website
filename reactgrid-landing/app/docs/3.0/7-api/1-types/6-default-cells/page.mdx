ReactGrid API contains a union of basic cells usually used for consuming changes and 
marking cells array inside the data row.

#### Definition

```ts
type DefaultCellTypes = CheckboxCell | DateCell | EmailCell | GroupCell | HeaderCell | NumberCell | TextCell | TimeCell;
```
